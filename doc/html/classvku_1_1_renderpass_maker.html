Detailed Description

See https://vulkan-tutorial.com for details of many operations here.

Factory for renderpasses. example: RenderpassMaker rpm; rpm.beginSubpass(vk::PipelineBindPoint::eGraphics); rpm.subpassColorAttachment(vk::ImageLayout::eColorAttachmentOptimal);

rpm.attachmentDescription(attachmentDesc); rpm.subpassDependency(dependency); s.renderPass_ = rpm.createUnique(device);
